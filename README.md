<h1 align="center">MultiGame-Hub</h1>

  <p align="center">
    A simple website which includes a Wordle clone which uses a dataset of all the 5 letter words given by Stanford University to check the validity of a word entered. This website also includes a Tic-Tac-Toe which has a Multiplayer to play with your friends and an AI which uses the minimax algorithmto play against you and tries to win.

[Learn more about Minimax Algorithm »](https://en.wikipedia.org/wiki/Minimax)

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/ItsBlankz/MultiGame-Hub.git
    ```
2. Install required libraries
    ```sh
    pip install -r /path/to/requirements.txt
    ```
3. Open terminal in your project folder
4. Run the following command
    ```sh
    flask run --reload --debug
    ```
5. By default the website should be hosted on the 5000 port but if u have set up an custom port please use the appropriate link

<h3 align="center"><a href="http://localhost:5000/">http://localhost:5000/</a></h3>
